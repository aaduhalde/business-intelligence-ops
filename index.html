Market Intelligence Data Warehouse

End-to-End Analytics Platform | SQL Server, Python, GitHub Actions

The Problem (The "Why")

Companies lack real-time visibility into job market trends and salary benchmarks, leading to slow and uninformed hiring or career decisions.

The Architecture (The "How")

Raw Data (SerpApi/Scraping) ➔ Python ETL ➔ Google Drive (Data Lake) ➔ SQL Server (Star Schema) ➔ Power BI

Data Modeling

Star Schema implementation with SCD Type 2 for historical salary tracking and Dimensional modeling for Skill-Gap analysis.

Key SQL Logic

WITH AvgSalary AS (
  SELECT role, AVG(salary) as avg_s 
  FROM fact_market_jobs
  GROUP BY role
) -- Window functions for IQ
